# CareerFlow

A production-ready full-stack application for tracking job applications and outreach workflows.

## Tech Stack

- Frontend: React + TypeScript (Vite)
- Backend: NestJS + TypeScript
- Database: PostgreSQL (Prisma ORM)
- CI/CD: GitHub Actions
- Deployment: Cloud (TBD)

## Project Goals

- Build a real-world, production-style application
- Follow clean architecture and best practices
- Implement automated testing and CI/CD
- Deploy and maintain the application
